SurfStatReadSurf1
==============================

Synopsis
---------

SurfStat's surface loader, used here for loading Freesurfer and .obj files. For surface loading in BrainSpace, please use :ref:`convert_surface.m` instead (`source code  <https://github.com/MICA-MNI/BrainSpace/blob/master/matlab/surface_manipulation/SurfStatReadSurf1.m>`_).
